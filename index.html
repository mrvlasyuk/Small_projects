GitHub Pages

Страница готова к статическому хостингу

Этот проект настроен для публикации на GitHub Pages. После пуша в репозиторий GitHub автоматически соберёт и задеплоит статические файлы.

  • Хостинг происходит из корня репозитория.
  • Вся логика статическая — достаточно HTML и CSS.
  • Изменения видны после очередного деплоя.
